Replot Network: Examples

Basic Example

In this example, we rely completely on Replot's smart defaults - simply plug in the data - and allow Replot to do the rest. We get a beautiful interactive network chart with just a few lines of code.

Custom Dimensions

The map can be displayed in custom dimensions using the height and width props.

Link Styles

Link color can be customized using the lineColor prop.

Link opacity can be customized using the lineOpacity prop.

Link width can be customized using the lineWidth prop.

Weighted Links

Link width can be weighted using the weightedLinks and linkKey props. If weightedLinks is specified, but linkKey is not, link width will be weighted by how many times the same link appears in the data.

Weighted links can be displayed in a custom range of width using the lineWidth and maxWidth props.

Node Colors

Nodes can be colored by groups using the nodes, nodeKey, and groupKey props.

Nodes can be colored with a custom palette using the color prop.

Node Radius

Node radius can be customized using the nodeRadius prop.

Weighted Nodes

Node radius can be weighted using the nodes, nodeKey, and nodeWeightKey props.

Weighted nodes can be displayed in a custom range of radius using the nodeRadius and maxRadius props.

Node Labels

Node labels can be switched on or off using the showLabels prop.

Node labels can be displayed in a custom color using the labelColor prop.

Node labels can display custom contents using the the nodes, nodeKey, and labelKey props.

Attraction Factors

Density of nodes can be customized using the attractionFactor prop.

Tooltip Features

The tooltip color can be switched between light and dark using the tooltipColor prop.

The tooltip can be switched on or off using the tooltip prop.

The tooltip can display custom contents using the tooltipContent prop.
